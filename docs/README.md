# Omega Project Documentation

This is the main documentation for the Omega cybersecurity ecosystem.

## Modules:

- **sentinel-agents**: Kernel hooks, memory-safe monitoring, agent runtime (Rust, C++, Zig)
- **mesh-network**: Distributed P2P messaging, consensus, event streaming (Go, Elixir)
- **intelligence-core**: GNNs, GANs, MARL, LLMs for predictive defense (Python, Julia)
- **wasm-modules**: Sandboxed runtime modules with cross-language compilation (Rust, WASM)
- **deception-engine**: Dynamic honey-reality and AR deception environments (Python, TypeScript, WebGPU)
- **digital-twin**: GPU-accelerated simulation of full networks and attacker/defender interactions (Unreal Engine, Python)
- **dashboard**: Real-time monitoring, visualization, AR/VR dashboards (TypeScript, WebGPU)
- **scripting**: Runtime agent self-mutation scripts (Lua)
- **orchestration**: High-concurrency orchestration and scheduling (Elixir, Scala)
- **trust-fabric**: Quantum-safe cryptography, blockchain DAG agent trust and integrity (Rust, Blockchain Layer)
- **human-threat-modeling**: Behavioral analysis, social engineering, attacker persona simulation (Python, LLMs)

More detailed documentation for each module will be provided within their respective directories.
